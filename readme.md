# Parse pixels

Загрузка и обработка Excel файлов со ссылками подготовленные к ручному копированию в различные форматы конфигураций и скриптов.

## Сборка

```bash
docker build -t parse-pixels .
```

## Запуск

```bash
docker run -p 8080:8080 parse-pixels
```

## Загрузка файла

Откройте в браузере адрес `http://localhost:8080` и загрузите файл с ссылками.
